# my-vim
## Setup
```
git clone https://github.com/trunliu/my-vim.git
chmod a+x setup.sh && ./setup.sh
```

## Usage
`C`-`Ctrl`
`A`-`Alt`

| Key | Note |
| :--: | :--: | 
| `C`+`]` | jump to the defination | 
| `C`+`o` | back | 
| `C`+`i` | unback  | 
| `u`|  undo | 
| `U` | redo | 
| `F1`   | open filetree the on the left |
| `F2`   | open function list on the right |
| `F3`   |  run `cflags -R` to regenerate tags |
